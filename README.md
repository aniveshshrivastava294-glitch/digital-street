# 🏪 Digital Street: The Neighborhood Marketplace

Welcome to **Digital Street**, a premium, state-of-the-art marketplace platform designed to revitalize local communities. We bridge the gap between neighborhood artisans, passionate farmers, and conscious consumers through a visually stunning and secure digital experience.

![Digital Street Hero](./admin_login_after_wait_1775022869287.png)

## ✨ key Features

*   **🎨 Premium 3D Aesthetic**: Experience immersive depth with GSAP-powered mouse parallax, 3D card tilts, and glassmorphic UI elements.
*   **💳 Secure Digital Wallet**: A robust, real-time wallet system for customers to manage funds and place orders instantly.
*   **🛒 Vendor Dashboard**: Empowering sellers with live shop toggles, inventory management, and one-click pickup verification via unique OTP codes.
*   **🛡️ Enterprise Admin Portal**: Comprehensive platform analytics including revenue tracking, order history, and customer support management.
*   **📱 Responsive First**: Fully optimized for Desktop, Tablet, and Mobile devices with fluid staggered animations on every viewport.

## 🛠️ Technology Stack

-   **Frontend**: Vanilla HTML5, CSS3 (with Modern Design Tokens)
-   **Animations**: [GSAP](https://greensock.com/gsap/) & [ScrollTrigger](https://greensock.com/scrolltrigger/)
-   **Backend**: [Firebase](https://firebase.google.com/) (Auth & Firestore)
-   **Icons**: [Lucide](https://lucide.dev/)
-   **Hosting**: Render (Frontend) / Firebase (Database)

## 🚀 Getting Started

### Prerequisites
-   A modern web browser.
-   Access to the project directory.

### Local Development
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/aniveshshrivastava294-glitch/digital-street.git
    ```
2.  **Open the project**:
    Simple open `frontend/index.html` in your browser, or use a local server like Live Server or `serve`:
    ```bash
    npx serve -s frontend
    ```
3.  **Firebase Config**:
    Ensure `frontend/firebase-config.js` is populated with your project's Firebase credentials.

## 👥 Roles
*   **Customer**: Browse products, manage wallet, and track orders.
*   **Vendor**: List products, toggle shop status (Live/Off), and verify customer pickups.
*   **Admin**: Oversee platform metrics and handle support requests.
